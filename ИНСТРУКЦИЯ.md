## Что нужно сделать (один раз)

### 1. Создать бота в Telegram

1. Открой Telegram → найди **@BotFather** (синяя галочка)
2. Напиши `/newbot`
3. Название: например `Воркшоп Женское Здоровье`
4. Username (должен заканчиваться на `bot`): например `zdorovye_elena_bot`
5. Скопируй **токен** — длинная строка вида `7123456789:AAF...`

### 2. Узнать свой Telegram ID

1. Найди бота **@userinfobot** → напиши `/start`
2. Он пришлёт число — это твой ID

### 3. Заполнить настройки в bot.py

Открой `bot.py` и замени в самом начале:

```python
BOT_TOKEN         = "вставь токен от BotFather"
ADMIN_IDS         = [вставь свой ID]
WORKSHOP_DATE_STR = "15 июля 2025, 10:00"
WORKSHOP_DATETIME = datetime(2025, 7, 15, 10, 0)
WORKSHOP_LOCATION = "https://t.me/ссылка_на_группу"
IP_INN            = "твой ИНН (12 цифр)"
IP_OGRNIP         = "твой ОГРНИП (15 цифр)"
IP_ADDRESS        = "юридический адрес ИП"
```

### 4. Положить QR-код рядом

Переименуй QR-код в **`qr_sbp.png`** и положи рядом с `bot.py`.

### 5. Установить и запустить

```bash
pip install -r requirements.txt
python bot.py
```

Всё! Папка с файлами:
```
📁 папка/
├── bot.py
├── requirements.txt
├── qr_sbp.png
└── bot.db  ← создастся автоматически при первом запуске
```

## Команды администратора

| Команда | Что делает |
|---------|-----------|
| `/participants` | Список участников + кнопка «Скачать Excel» |
| `/export` | Сразу скачать Excel-файл |
| `/stats` | Статистика: кол-во, сумма |
| `/broadcast` | Рассылка (выбор: оплатившим / всем) |
| `/admin` | Справка |
| `/confirm_ID_СУММА` | Подтвердить оплату + отправить чек |
| `/reply_ID текст` | Ответить участнице на вопрос |

### Подтверждение оплаты:
Когда придёт уведомление со скриншотом, в тексте уже будет готовая команда.
Просто нажми на неё и отправь. Например: `/confirm_123456789_10000`

Бот сам обновит базу и отправит чек участнице.

### Рассылка ссылки на группу:
```
/broadcast → Только оплатившим
→ напиши текст: 🌸 Ссылка на группу воркшопа: https://t.me/...
```

## Запуск 24/7 + автоматический доступ после оплаты на сайте

Деплой на **Railway.app** (как web-сервис — чтобы был публичный адрес для вебхука):
1. Зайди на https://railway.app → войди через GitHub → New Project → Deploy from repo.
2. На вкладке **Variables** задай переменные (см. `.env.example`):
   - `BOT_TOKEN` — новый токен от @BotFather (старый отзови через /revoke!)
   - `ADMIN_IDS` — твой Telegram ID (несколько — через запятую)
   - `TILDA_WEBHOOK_SECRET` — придуманная длинная строка-секрет
   - `WORKSHOP_CHANNEL_ID` — ID закрытого канала (вида -100…), бот — админ канала
3. Railway выдаст домен вида `https://…up.railway.app`.

### Настройка вебхука в Тильде
В настройках приёма заявок/оплаты Тильды укажи URL:
```
https://ТВОЙ-ДОМЕН.up.railway.app/tilda-webhook?secret=ТВОЙ_СЕКРЕТ
```

### Как работает автодоступ
- Человек регистрируется в боте (email + телефон) и оплачивает на сайте → вебхук
  находит его по email/телефону → бот сам присылает чек и персональную ссылку в канал.
- Если он **оплатил раньше**, чем зашёл в бота — оплата сохраняется, и доступ выдаётся
  автоматически в момент регистрации (по совпадению email или телефона).
- Неоплатившие ссылку в канал не получают — она отправляется только после подтверждения оплаты.

⚠️ Условие: email/телефон в форме оплаты Тильды должны совпадать с указанными в боте.
